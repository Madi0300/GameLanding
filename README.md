# Game Landing

Лендинг для вымышленного social battle royale (стилизация под игру SOS): промо-экран с CTA, описание, карусель скриншотов, список фич, таблица системных требований, пресс‑цитаты и форма подписки.

## Live demo
https://madi0300.github.io/GameLanding/

## Особенности
- Хиро‑блок с кнопкой «Buy Now on Steam», ценником и якорем «The story» для плавного скролла вниз.
- Блок «What is SOS?» с галереей скриншотов и переключением по стрелке.
- Раздел «features» с краткими преимуществами и визуальным акцентом на первом пункте.
- Таблица «system requirements» с разделением по ОС, CPU, RAM, диску и видеокарте.
- «Press quotes» с карточками отзывов, датами и ссылками на соцсети.
- Форма подписки на новости с плейсхолдером и акцентной кнопкой.

## Технологии
- Чистый HTML/CSS, основной стиль в `css/style.css`, reset в `css/default.css`.
- Исходный SCSS в `#sources/style.scss` (сборка через prepros.config).
- Vanilla JS: плавный скролл (js/script.js), выпадающий выбор языка в шапке, управление высотой hero и позициями элементов при ресайзе (js/enableStyles.js), логика слайдера для галереи (js/galeryLogic.js).

## Как запустить локально
1. Склонируйте проект или скачайте архив.
2. Откройте `index.html` напрямую в браузере **или** поднимите статический сервер (например, `npx serve .` в корне).

## Структура
- `index.html` — разметка страницы.
- `css/` — стили (`default.css`, `style.css`).
- `js/` — скрипты для интерактивных элементов.
- `img/`, `fonts/` — медиа и шрифты.
- `#sources/style.scss` — исходник стилей для препроцессора.
